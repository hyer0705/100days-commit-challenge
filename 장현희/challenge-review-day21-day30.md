# 10일 동안 공부한 내용
- 파일 저장할 때 파일을 RequestParam이나 RequestPart로 전송 가능
- MultipartFile로 파일 데이터 받고, 테스트에서 MockMultipartFile로 파일을 임의로 만들 수 있다.

- s3: 정적 파일 저장하는 aws 서비스(확장성이 좋고, rds보다 비용 저렴)
- elasticache: redis를 위한 aws 서비스
